学习 YOLO

一个格子两个预测框，每个预测框用5 个参数，四个是 xywh，x 是预测框 bounding box 的中心点距离格子 cell 左边的距离，y 是中心点距离 cell 上边的距离（注意，一张图片分割出来的 49 个 cell 的格子都是一样大的），w 是预测框的宽度（和 x 同向），h 是高度，所以归一化的时候，是 x=x/wc，y=y/hc，wc 和 hc 分别指的是格子的宽度和高度，h 和 w 是整个图像的高度和宽度。
然后每个格子还需要一组类别值，注意只有一组，虽然是两个预测框，但他们共享一组类别概率值值，比如 VOC 数据集20 个类别，那就有 20 个类别概率值，分别代表这个格子属于这个类别的概率值，进一步说，使用这一组 20 个概率值来形容这个格子属于哪一个类。
不管一个单元格预测了多少个 bounding box，一个单元格也只能预测一组类别概率值。
对一张图片我们切割成 77=49 个格子，所以一张图片的预测结果一共是 77*30 个 tensor(张量、或者说参数)。

每个预测框都有两个参数，第一个是 Pr(Object) 表示这个框中是否有物体，也就是 0 和 1，第二个是这个框的 IOU，即交并比，预测框和 GT 框的交集大小除以并集大小，也就是一个预测准度。
测试的时候，我们将这二者还有刚刚那 20 个概率值的任一一个分别相乘来得到 这个预测框是这个类别的概率和框包含物体的准度。即 **特定类出现在 cell 内的概率和位置匹配程度(边框拟合程度)**，感觉不要太区分 cell 和 boundingbox 的区别。

最终，每个格子只能预测一个类别，所以限制了 yolo 预测重叠或临近物体的数量（比如鸟群中的鸟），yolo 只能预测 49 个对象。

每个 cell 虽然有两个框，但是是两个中取一个，框的粗细表示了可信度，选择可行度高的那一个。

卷积

卷积就是拿一个卷积核（模板）去找，让他匹配计算，如果匹配到一个和模板很像的区域，就会得到一个大值。
卷积结果叫特征图 feature map（或者叫特征图像），是对图像是否包含这个特征的一种表示，包含的话包含在哪，那就是比较大的那个值。特征图像是小于等于原始图像的。所以想要有效果，卷积核一定要好好设置。

全连接层

用来提取特征，举例子这里通过卷积提取了 H、O两个特征，全连接指的就是前面的H、O和和后面的 H2O、O2、H2 是全连接上的（每一个神经元和下边的神经元都是连接上的），并且全连接中还有权重，有时候我们会进行 dropout（随机失活，让一些神经元不可用，来防止过拟合）。

　　(2)False positives(FP): 被错误地划分为正例的个数，即实际为负例但被分类器划分为正例的实例数；

1.精确率（precision）

precision = TP / ( TP + FP )

精确率是精确性的指标，表示被分类器正确分为正例的个数(TP)占被分类器分为正例的样本(TP+FP)的比重。

2.召回率（recall）

recall = TP / ( TP + FN ） = TP / P

召回率是覆盖面的度量，也就是被分类器正确分为正例的个数(TP)占原始数据中全部正例(TP+FN)的比重。

只有当P和R都很高的时候，F1(用准确度和召回率算的)才会高，所以称为调和平均数，F1的取值范围是0到1。

5.accuracy

准确率，这个跟精确率只有一字之差，但实际上有很大的不同，它是指正确分类（不管是正确分为P还是N）的比率

accuray = （TP + TN) / (TP + FP + TN + FN ) = (TP + TN) / (P+N)

Precision衡量你的预测有多准确。也就是说，你的预测正确的百分比。
Recall衡量您发现所有正例的能力。 例如，我们可以在前K个预测中找到80％的正例。（TopK）

AP 平均精度 就是 PR 曲线的面积。mAP 就是 所有类别的 AP 的平均值。
为什么要用AP或者mAP
我们希望一个模型的Precision和Recall都很高，所以需要综合考虑这两个因素，我们可以联想到用调和平均数F1-beta值来衡量，另一种方法正是PR曲线下的面积AUC，这也就是AP。AUC面积越接近1性能越好。曲线下的面积理解为不同召回值的情况下所有精度的平均值。

自注意力机制
ViT不直接看像素，而是对图像进行切分，比如224*224的切成14*14个方块，每个方块叫token，每一层中，每个方块会向所有方块询问，比如最左边一个快是垫片，他询问谁和我有逻辑关系。，此时图像最右边的token回答我是螺栓和你是一套的，因此即便距离很远，模型给他们之间连上高权重的线，从而信息交换。而CNN要实现这个最左最有的通信可能需要几十层的卷积，中间信息模糊了。
在铁路场景，可以借助长距离依赖来实现对称性检查，比如左右各一个弹簧，左边坏了，CNN很难通过对比来发现异常，而ViT 同时看两个，发现二者不一样。 此外如果我们要求物件在某个背景下才算是正常，自注意力也可以把背景信息编码进零件的特征。
ViT具有全局视角

卷积核通常很小，感受野很小，不下采样的话，卷积后的模型始终只能看到局部，下采样相当于退后一步看全身，减少图像尺寸，同样的卷积核能覆盖到原图更大比例的面积。同时也能减少计算压力，去除冗余信息。
下采样通常通过池化或者步长卷积来实现，池化比如最大池化，步长卷积指的是卷积核跳着走，比如步长2，扫完一个区域之后跳过一个像素再扫，这样特征图长宽就减半了。

我们使用跳跃连接把分辨率救回来，它相当于从前端（浅层特征，知道位置，但不明白语义）即哪些还没模糊的高清特征图直接投送到后端（深层特征，具备语义，你知道这个点有螺栓那里有裂纹但你不知道具体位置）。
具体是，将深层7*7的特征图上采样（反卷积或者插值）便会224*224特征图，虽然够大但是模糊，然后这张图和那张浅层图叠在一起，浅层图告诉模型这里有个东西（位置信息），深层图告诉模型这个是什么（语义信息）。

反卷积：指的是有参数的填充放大，将小尺寸特征图填充0放大，然后学习一个卷积核去扫这张撑开的图，学习去用合理的数值填充这些0，使得结果清晰又合理，由于卷积核带参数，效果比简单的插值放大更合理。

FPN 特征金字塔就是跳跃连接的升级版，能同时检测巨大的形变和微小的划痕，是目标检测和异常检测的关键。
区别在哪，跳跃连接 U-Net 通常只在最后的一层高分辨率特征上进行预测，应该是说 低分辨率 特征上采样之后和高分辨率特征融合，只用这一层进行预测。而FPN 是多层预测，在金字塔的每一个层级都分别进行检测，比如 p2 层专门检测10像素的小裂纹，而p5层专门检测半张图的大型变形。还提到“似乎每一层的信息是实时共享的”？如何理解

FPN 的特点应该是，它在每一层特征都做了跳跃连接。
FPN 的构造分成三个路径，第一个 自下而上，是特征提取过程，就是之前说的下采样，特征图分辨率越来越低，语义越来越强，能认得出东西。 第二个是 自上而下，把深层特征进行上采样放大，试图让他回到高分辨率的状态。 第三个是 横向连接，他将第一步路径中 每个尺寸的特征图通过 1*1 卷积处理后 直接加到 第二步路径中的特征图，从而在每一个尺度上，都生成了一个 具有高清分辨率和深度语义的特征图。
那么 FPN 它的特点就是，它的顶层特征（感受野大）能去识别大物体，底层特征（分辨率高）能去识别小的痕迹。
通过多层特征融合，增强了模型对噪声的鲁棒性，即使某一层特征不明显，其他层的融合信息也能一定程度上起到补位的作用。
还提到一个 语义对齐，比如我们要判断某个小物体 螺栓 不仅正常，也处于某个正确的位置，没有松动，我们需要模型唔够获取周围的结构信息，这就属于 深层语义，FPN 的横向连接就是让每一个小尺度特征都带上 全局的“脑子”，让它能获取不仅限于这个小尺度的语义信息？

上面提到的 1*1 卷积核，他又很大的作用，第一个 对齐通道数，降为或升维，自下而上，浅层（分辨率高）通道数小，深层的通道数可能到了512，如果希望将两张特征图叠加，那要求通道数必须一致，1*1 卷积核就能将512的深层特征压缩为64通道，将同一个位置不同通道的信息进行组合。

再讲一遍：
第一阶段 自下而上提炼特征：p2浅层尺寸大（注意理解这个尺寸），分辨率高，语义差，升级到p5深层，尺寸小（下采样很多次），能看到整体布局，认出这个螺栓应该在的位置（在转向架的左边），也就是语义强，但是它看不清楚螺栓，只是知道它在那里。
第二阶段 自上而下传递智慧，将p5深层特征反卷积放大到和p4一样大，此时，这张大图仍然具有较为强大的语义信息，具备“全局意识”。
第三阶段：一对一融合，将放大后的p5（和4一样大，但具有5的大局观）和原来的p4（有原来的细节）进行像素级相加，融合得到的特征图，每一个像素点既包含原来p4的高分辨率位置细节也包含了p5传下来的语义信息。

举个例子：假如我在p2（分辨率很高的一层），我现在看到一个黑点（我很适合用来看它具体的位置），我认不出是螺栓上的裂纹还是钢轨上的油渍，但是有了FPN，深层（p5、p4、p3）的语义信息一层一层向下传给了p2，现在p2的哪个点，他携带的信息就成了“我是位于 5：转向架、4：左侧第三个 3：螺栓上的裂纹”，从而获取到不属于自己尺度的语义信息，它们从金字塔顶端顺着 自上而下 的路径留下来，和每一层的局部细节汇和，得到的每一层都是有脑子的高清图，可以在每一层都发起检测。

为什么尺寸相同，从p3卷积而来的p4和从p5放大回来的p4，一个有语义，一个有细节？
前者的每一个像素点，“只看过”原图中的一小区域，比如16*16像素，他能看的清螺纹上的每一个沟壑，具体到位置，但它不知道这个螺纹是装在车顶还是车轮，也就是有细节，没语义。
而后者，它从p5而来，p5是经过多次下采样的，它看过原图的很大部分，比如128乘128像素，他理解了复杂的逻辑关系，比如这颗螺纹属于转向架，我们通过反卷积得到的p4也具备了这种能力，同样的，它也和p5类似（原本的螺纹细节在下采样的时候就被丢失了，使用的反采样无法恢复这种细节），缺少了细节。
我们需要两张特征图具有一样的大小，才能进行融合，一个高清局部图，一个是模糊的全局逻辑图，两者融合就是 逻辑套在细节 上。

举个例子学下 1乘1 卷积？
比如一个特征图三个通道，分别代表了三种语义判断，
通道1：这里的颜色像金属
通道2：这里的形状像圆形
通道3：这里的纹理像铁锈
我们想得到一个最终判断：这里是不是一个生锈的螺栓，靠任何一个通道都不行，必须通过1乘1卷积核融合起来，它就像一个加权系数，每个通道都对应了一个权重。
结果=0.4金属+0.5圆形+0.8铁锈
它实现了 1、属性整合成语义， 2、降维，浅层的c4有256个通道，深层的c5有512个通道，他们不能直接相加，于是我们用256个1乘1卷积核去扫c5，把512层的厚信息拍扁成126层，它不仅筛选出重要的语义给下一层，还压缩了尺寸。

理解特征图的通道：
图的长宽代表位置，通道代表 “属性”
输入层，原始照片有三个通道RGB，每个通道代表每个像素点的颜色属性
其他层，随着卷积的进行，通道开始具有其他语义特征，比如
通道1 负责寻找 垂直边缘
通道2 负责寻找 金属反光
通道 100负责寻找 类似螺栓的圆形状
可以把通道理解为 观察图像的视角，一个512通道的特征图意味着模型从512个不同维度来审视特征图的每个像素点所对应的 原图区域（这块区域也叫做感受野，比如在浅层，一个特征点对应的是原图3x3的区域，而在深层，一个点可能对应原图的128x128的区域）。

卷积过程中为什么长宽和通道数可以同时变？
卷积核也是有厚度（通道）的，靠卷积核的数量实现通道数的变化，靠卷积的步长和池化实现长宽变化。比如：
输入 224乘224乘3
第一层卷积过程：使用的卷积核的通道数等于输入图的通道数，假如用 3乘3乘3的核扫过图片（这个立方体卷积核的27个权重分别和图像局部区域的27个值对应相乘然后全部累加，可能还要加个偏置最终得到一个数值），得到一个224乘224乘1的特征图，使用64个这样的卷积核就能得到64个特征图，叠加后就是64通道的特征图，而长宽的变化可以依靠卷积步长比如2的步长（长宽直接减半，图像变小了，但是每个点涵盖范围更大了），或者池化 直接缩小。

特征图融合是直接相加吗？
相加是有前提的：
1、通道比如对齐，不只是大小对应，从512通道的图翻译到256通道的过程，是会不断学习的
2、相加的本质是特征增强，浅层图的通道A检测到 一根细线，深层图的 通道A 检测到这里该有螺栓边缘，相加之后，两个信号都变强，模型就会判定这里是一个精准的螺栓边缘。
好比：深层特征认为这个范围有个坑，浅层特征认为这里有黑影，二者重合就是很好的结论了。

深度学习的强大之处在于将一块区域复杂的信息浓缩成512个有意义的数字，在异常检测中，如果这个数字的组合与模型见过的正常组合不一样则会报警。

鉴于特征重建任务，我们得出结论，学生网络不应复制教师网络的架构。
我们注意到所提的任务设计类似于图像去噪，但我们希望在特征空间中对图像进行去噪。编码-解码器架构被广泛用于图像去噪。因此，我们将它作为去噪学生网络的架构采用。

讲了不同插值

在 [31， 26] 中，多层特征的余弦距离被直接相加表示每个像素的异常分数。然而，如果所有层级特征的区分不均等准确，结果可能不尽理想。为解决这个问题，我们增加了分段网络，通过额外的监督信号引导特征融合。
我们冻结学生和教师网络的权重，以训练细分网络。合成异常图像作为两个 S-T 网络的输入，对应的二元异常掩码即为真实数据。
三个尺度的特征图各自逐元素归一化乘积然后上采样到特征图1的大小，最后拼接作为分割网络的输入。分段网络包含两个残差块和一个 Atrous 空间金字塔池（ASPP）模块 [5]。没有上采样或下采样;因此，输出大小等于的输入的大小（原图的四分之一）。虽然这可能导致一定程度的分辨率损失，但降低了训练和推理的记忆成本，而这在实际作中至关重要。

分割训练通过利用焦损 [17] 和 L1 损耗进行优化。在训练集中，大多数像素是正常的，容易识别为背景。图像中只有一小部分是异常像素，需要被分割。因此，焦点损失有助于模型聚焦于少数类和难样本。此外，L1 损耗被用来改善输出的稀疏性，使分割掩码的边界更加清晰。计算损耗时，我们将真实异常掩码下采样到输入大小的四分之一，与输出匹配。

在推断阶段，测试图像会同时输入教师和学生网络。最终，分割后的预测被上采样到输入大小，并作为异常分数图。输入图像中的异常像素预期在输出中会有更高的数值。为了计算图像级异常得分，我们使用异常分数图中最大 T值的平均值，其中 T是调优超参数。
可描述纹理数据集（DTD） [6] 作为方程 1 中的 异常源图像 A。 [36] 显示其他数据集如 ImageNet 也能实现类似性能，但 DTD 体积更小且易于使用。

我们注意到之前的像素级评估实现对齐不佳。大多数研究将二值真图下采样加快计算速度，但也有部分进行了额外的中心裁剪 [24， 8， 7]。 此外，降采样实现也未标准化导致基准真值变化且评估不公平。在某些情况下，降采样会引入严重失真。我们的工作中，我们使用双线性插值将二进制掩模下采样到256，然后将结果四舍五入，阈值为 0.5。这种实现可以保持原始地面真实掩码的连续性，避免高估或低估。

是降噪网络用来消除异常特征里面的噪声成为正常特征。

一类在图像层级上生成人造异常图像，如CutPaste[24]方法通过在图像中随机复制一个矩形区域后将其随机粘贴于该图片上以模拟异常，Draem[28]方法通过二维柏林噪声[79]来模拟异常应该生成的位置，RealNet[46]方法则尝试通过使用一个强度可控的扩散模型来生成更为真实的异常图像。另一类方法则选择在深度特征层面模拟图像中的异常在特征层的影响，如DSR[80]方法和SimpleNet[81]方法都属于此类别。设定以0.8的概率生成人造异常，也就是说模型在训练过程中遇到的图像可能是正常的，也可能是含有人造异常的，以此来防止模型过拟合。此异常生成方法相比RealNet[46]中采用扩散模型的方法，更加节省内存资源与计算资源，相比CutPaste[24]方法中生成规则的矩形异常，此类方法生成的异常形状不定，更加贴近真实异常，相比特征层级的异常生成策略，此类方法则可以生成相应的掩码M，可用于指导模型进行训练。

RD4AD[26]采用了反向蒸馏的方法来增强模型对异常样本的识别能力，即将教师网络的输出作为学生网络的输入，本模型也将此思想应用到了深度特征生成网络与深度特征降噪网络中。

对于异常检测任务来说，其主要目的为学习到如何检测异常像素所在的位置，因此通道级的注意力机制在检测异常方面是无效的[76]，因此在此采用像素级的注意力机制。该注意力机制接收深度特征生成网络的输出，并将其聚合为一个通道，用以向学生网络传输提示以促进学习过程。在此通过一个像素级注意力机制，向深度特征降噪网络指导本层特征图中的重要的异常像素位置，从而帮助降噪网络准确的将异常特征重建为正常特征。

仅使用最深层的深度特征进行建模时（可以理解为从深度特征恢复出原图），常常会导致微小异常及异常边缘部分的判断失误。因此为提高模型对微小异常及边缘部分的检测，需要我们对多尺度的深度特征进行融合，充分利用不同层级上多尺度特征各自的信息。受EfficientDet[83]模型启发，在此提出基于可学习权重的特征聚合网络用以对深度特征生成网络的多层特征进行聚合。

不同的输入特征具有不同的分辨率，通常情况下他们对输出特征的贡献也不相同，所以在此为每个输入添加了一个额外的可学习的权重，从而让网络自身学习每个输入特征的重要性，以实现自适应的特征聚合。

在进行异常定位时，常采用计算教师网络和学生网络多尺度特征之间的距离来实现[84][85][86][87]。然而，如果不同层级特征的质量不均匀，那么这种方法可能会产生次优的结果。因此，本章引入了DeSTSeg[29]中提出的分割网络来指导最终像素级异常的判定。将深度特征生成网络的输出和深度特征降噪网络的解码器的输出通过双线性插值将他们上采样到图像大小后计算特征对应像素的乘积（和DeSTSeg一样），再将多尺度特征在通道维度上进行特征拼接后，作为分割网络的输入。
该分割网络由两个残差块后接着一个ASPP(Atrous Spatial Pyramid Pooling)[88]模块组成，其损失函数采用焦点损失[89]和L1损失之和。

RD4AD

以往研究中使用相似或相同的架构构建教师和学生模型，会阻碍异常表征的多样性。为解决这一问题，我们提出了一种新的 T-S 模型，由教师编码器和学生解码器组成，并相应引入一个简单但有效的“逆蒸馏”范式。学生网络不再直接接收原始图像，而是以教师模型的单类embedding(意指翻译和浓缩)为输入，并以恢复教师的多尺度表示为目标。本研究的知识蒸馏本质上从抽象的高级呈现开始，逐步到低层特征。此外，我们在 T-S 模型中引入了可训练的单类瓶颈浓缩（OCBE）模块。得到的紧凑浓缩有效保留了关于正常模式的重要信息，但放弃了异常扰动。

在知识蒸馏（KD）[6， 15]中，知识在师生（T-S）对 内传递。在无监督式 AD 的背景下，由于学生在训练中只体验到正常样本，当查询异常时，二者可能会产生不一致的表征。这一假设构成了基于 KD 的异常检测方法的基础。然而，由于（1）师生网络的架构完全相同或相似（即不区分过滤器[33]）以及（2）知识转移/提炼过程中 T-S 模型的**数据流(应该是指编码和解码的卷积过程)**相同，这一假设在实际中并不总是成立。虽然使用较小的学生网络部分解决了这一问题[33， 39]，但浅层架构较弱的表示能力阻碍了模型准确检测和定位异常。

ps：深度学习的 低维特征(low-dim Embedding) 通常指的是深层特征，比如卷积了很多次之后的网络的瓶颈层，因为这里的维度指的是 空间维度，不是通道数。
首先，与传统的知识蒸馏框架不同，传统中教师和学生都采用编码器结构，在我们的逆向蒸馏的T-S 模型由异构架构组成：教师编码器和学生解码器。其次，学生解码器的输入并不是原始图像数据，而是将老师压缩的得到的低维特征作为输入，通过恢复出教师模型在不同尺度得到的特征 从而来模拟教师的行为。从回归的角度来看，我们的逆向蒸馏利用学生网络来预测教师模型的特征。因此，这里的“反向”既指教师编码器与学生译码器的反向形状，也指的是先提炼高层表示，随后提炼低层特征的独特知识提炼顺序。


两个显著优势：i）结构不相似。在提出的 T-S 模型中 ，可以将教师编码器视为下采样滤波器，学生译码器视为上采样滤波器。“反向结构”避免了我们上文讨论的非区分滤波器造成的混淆[33]。ii）紧致性特征。输入给学生解码器的低维特征作为正常模式恢复的信息瓶颈。我们将异常特征表述为对正常模式的扰动。紧致特征有助于防止此类异常扰动传播到学生模型，从而增强 T-S 模型在异常上的表示差异。
由于瓶颈嵌入的紧凑性对异常检测至关重要（如上所述），我们引入了一个单类瓶颈嵌入（OCBE）模块，进一步压缩特征代码。我们的 OCBE 模块包括多尺度特征融合（MFF）模块和单类嵌入（OCE）模块，两者均与学生解码器共同优化。值得注意的是，前者将低层和高层特征聚合，构建了丰富的嵌入用于正常模式重建，后者旨在保留有利于学生解读教师的响应的关键信息。
基于图像重建的方法（AE、GAN）依赖于这样一个假设：仅在正常样本上训练的生成模型能够成功重建无异常区域，但对异常区域则失败 [2， 5， 34]。然而，最新研究表明，深度模型的泛化能力极佳，即使是异常区域也能被很好地恢复[46]。为解决这一问题，基于重建的方法中融入了记忆机制[11， 16， 26]、图像掩蔽策略[42， 46]和伪异常[28， 45]。然而，这些方法在现实世界异常检测方面仍缺乏强的区分能力[3， 5]。
数据增强策略也被广泛使用。通过在提供的无异常样本中添加伪异常，无监督任务被转换为有监督学习任务[23， 42， 46]。然而，这些方法容易偏向伪异常值，无法检测到多种异常类型。例如，CutPaste [23] 通过在正常图像上添加小块来生成伪离群值，并训练模型检测这些异常区域。由于模型侧重于检测边缘不连续和纹理扰动等局部特征，因此未能检测和定位如图 6 所示的大型缺陷和全局结构异常。
从预训练模型中提取知识是异常检测的另一种潜在解决方案。在无监督式 AD 的背景下，由于学生模型在知识蒸馏中接触到无异常样本，T-S 模型在推理异常时应产生不一致特征[4,33,39]。为了进一步提升 T-S 模型对各种异常类型的辨别能力，引入了不同的策略。例如，为了捕捉多尺度异常，[4]集合了多个在不同尺度正规数据上训练的模型，MKD [33]则建议使用多层次特征比对。

教师 E 会提取多尺度表示。我们计划培训一名学生 D，恢复瓶颈嵌入中的特征。在推断过程中，教师 E 提取的特征可以捕捉异常样本中异常、分布外的特征。然而，学生解码器 D 无法从相应嵌入中重建这些异常特征。如果异常图像在拟议的 T-S 模型中产生的异常表征相似度低，表明异常评分较高。我们认为，异构编码器和解码器结构以及反向知识提炼顺序对异常表示的差异贡献很大。此外，可训练的 OCBE 模块进一步将多尺度模式压缩到极低维空间，用于下游的正态表示重建。这进一步改善了我们 T-S 模型中产生的异常特征的差异，因为教师模型产生的异常表示很可能被 OCBE 丢弃。为了进一步促进一类提炼，我们设计了一个可训练的 OCEB 模块，连接师生模式。
在传统的 KD 中，学生网络采用与教师模型相似或相同的神经网络，接受原始数据/图像作为输入，并针对其特征激活想与教师的成功匹配[4， 33]。在无监督 AD 的单类提炼背景下，当查询为异常样本时，学生模型预期会产生教师截然不同的表征[11， 26]。然而，异常上的激活差异有时会消失，导致异常检测失败。我们认为，这一问题归因于教师和学生网络的相似架构以及 T-S 知识转移过程中相同的数据流。

我们在消融研究中表明，ResNet [14] 和 WideResNet [44] 都是很好的候选者，因为它们能够从图像中提取丰富的特征 [4， 8， 23， 29]。学生解码器 D 的架构和老师的是对称的，但与 E 相反。反向设计有助于消除学生网络对异常的反应，而对称性则使其具有与教师网络相同的表征维度。例如，当我们以 ResNet 为教师模型时，学生 D 由多个类似残差的镜像对称解码块组成。具体来说，ResNet 中的降采样通过一个核大小为 1、步幅为 2 的卷积层实现[14]。学生 D 中对应的译码块采用了核大小为 2、步幅为 2 的反卷积层[47]。

T-S 模型在中低层和高层特征的低相似性分别表明 局部异常 和 区域/全局结构异常值。在 T-S 模型中，余弦相似度被视为 KD 损失，因为这样更精确地捕捉高维和低维信息中的关系[37， 49]。具体来说，对于学生和老师第k层的特征张量，我们计算它们沿通道轴的向量的余弦相似度损失，得到一个二维的异常图 M，M某个位置值比较大意味着该处异常大。

以直接将骨干中最后一个编码块的激活输出给 解码块D。然而，这种天真的联系存在两个不足。首先，KD 的教师模式通常具备较高的容量。尽管高容量模型有助于提取丰富的特征，但获得的高维描述符可能存在相当大的冗余。表征的高自由度和冗余性对学生模型解码无异常特征的核心是有害的。其次，骨干中最后一个编码块的激活通常能描述输入数据的语义和结构信息。由于知识蒸馏的顺序相反，直接将高层表示输入学生解码器对低层特征重建构成了挑战。

我们引入了一个可训练的单类嵌入块，将教师模型的高维表示投影到低维空间中。 我们将异常特征表述为对正常模式的扰动。紧嵌入则成为信息的瓶颈来防止异常扰动传播到学生模型，从而增强 T-S 模型在异常上的表示差异。本研究采用 ResNet 的第四层残留块[14]作为单类嵌入块（OCE）。

为了解决解码器低层特征恢复的问题，MFF 块在单类嵌入前将多尺度表示串接起来。为了在特征连接中实现表示对齐，我们对浅层特征进行一个或多个3x3、步幅为 2的卷积层下采样，随后进行批量归一化和 ReLU 激活函数。然后利用步幅为 1 的 1×1 卷积层和带有 relu 激活的批量归一化，获得丰富而紧凑的特征。

SimpleNet

目前的方法以无监督方式解决该问题，即在训练过程中仅使用正常样本。基于重建的方法[10， 21， 31]、基于综合的方法[17， 30]和基于嵌入的方法[6， 22， 24]是解决该问题的三大主要趋势。
基于重建的方法如[21,31]假设仅用正常数据训练的深度网络无法准确重建异常区域。像素级重建误差被视为异常定位的异常分数。然而，这一假设并不总是成立，有时网络能够“泛化”得非常好，以至于能很好地重建异常输入，从而导致误检[10， 19]。基于合成的方法[17， 30]通过对无异常图像生成的合成异常进行训练，估计正常与异常之间的决策边界。然而，合成图像并不够真实。合成数据中的特征可能偏离正态特征，使用此类负样本训练可能导致松散界限的正态特征空间，意味着不明确缺陷可能被包含在分布特征空间中。

近年来，基于嵌入的方法[6， 7， 22， 24]实现了最先进的性能。这些方法使用 ImageNet 预训练卷积神经网络（CNN）提取广义的正规特征。然后采用多元高斯分布[6]、归一化流[24]和记忆库[22]等统计算法来嵌入正态特征分布。异常通过比较输入特征与学习到的分布或记忆特征来检测。然而，工业图像通常与 ImageNet 的分布不同。直接使用这些偏置特征可能导致不匹配问题。此外，统计算法总是面临高计算复杂度或高内存消耗的问题。

SimpleNet 利用了基于合成和嵌入的方式，并进行了多项改进。首先，我们提议使用特征适配器生成面向目标的特征，从而减少领域偏置。其次，我们提出通过对特征空间中的正常特征进行噪声，而不是直接在图像上合成异常。我们认为，通过对噪声进行适当校准，可以获得一个紧密有界的正常特征空间。第三，我们通过训练一个简单的判别器来简化异常检测过程，这比上述基于嵌入的方法采用的复杂统计算法计算效率更高。具体来说，SimpleNet 利用预训练的骨干链进行正常特征提取，随后通过特征适配器将特征转移到目标域。然后，通过在相应的正常特征中加入高斯噪声，简单生成异常特征。一个由几层 MLP 组成的简单判别器会在这些特征上训练以区分异常。

基于重建的方法
异常图像区域不应被正确重建，因为它们不存在于训练样本中。一些方法[10]利用生成模型，如自动编码器和生成对抗网络[11]来编码和重建正常数据。其他方法[13,21,31]将异常检测视为修复问题，即图像中随机遮罩补丁。然后，利用神经网络预测被删除的信息。结构相似度指数（SSIM）[29]损失函数在训练中被广泛使用。异常图是通过输入图像与重建图像之间的像素差生成的。然而，如果异常与正常训练数据共享共同的组成模式（如局部边缘），或者解码器“过强”能很好地解码某些异常编码，那么图像中的异常很可能能够被良好重建[31]。

基于合成的方法
DRÆM [30] 提出一个网络，让网络像做判断题一样（discriminatively），端到端地去训练，强迫学会区分‘真正常’和很难分辨的‘伪异常’。CutPaste [17] 提出了一种简单的策略，用于生成合成异常，通过剪切图像补丁并在大图像的随机位置粘贴。CNN 经过训练从正态和增强数据分布区分图像。然而，合成异常的外观与真实异常并不完全一致。实际上，由于缺陷多样且不可预测，生成包含所有离群值的异常集是不可能的。与其在图像上合成异常，我们提出的的 SimpleNet，是在特征空间合成负样本。

PatchCore： 记住了正常特征的样本（建立一个巨大的数据库，存下来）。

基于嵌入的方法
近年来取得了最先进的性能。这些方法将正常特征嵌入压缩空间中。异常特征远离嵌入空间中的正常群。典型方法[6,7,22,24]利用在 ImageNet 上预训练的网络进行特征提取。通过预训练模型，PaDiM [6] 通过在多元高斯分布嵌入种提取的异常图像块特征。PatchCore 使用了一个最具代表性的、由正常图像块特征组成的记忆库。测试中采用马哈拉诺比斯距离或最大特征距离来评分输入特征。然而，工业图像通常与 ImageNet 的分布不同。直接使用预训练特征可能导致不匹配问题。此外，无论是计算协方差逆[6]还是在内存库中搜索最近邻[22]，都会限制实时性能，尤其是对边缘设备而言。

CS-Flow [24]、CFLOW-AD [12]和 DifferNet [23]提出通过归一化流（NF）将正常特征分布转换为高斯分布[20]。由于归一化流只能处理全尺寸特征图，即不允许下采样，且耦合层[9]占用的内存比普通卷积层多几倍，因此这些方法占用内存。蒸馏方法[4， 7]训练学生网络，使其与仅有正常样本的固定预训练教师网络的输出匹配。对于异常查询，应检测到学生和教师输出之间的差异。

SimpleNet 克服了上述问题。SimpleNet 使用特征适配器对目标数据集进行迁移学习，以减轻预训练 CNN 的偏向，SimpleNet 建议在特征空间中合成异常，而非直接在图像上。SimpleNet 采用单流推理方式，完全由传统 CNN 模块构建，便于快速训练、推理和工业应用。

SimpleNet 由特征提取器、特征适配器、异常特征生成器和判别器组成。异常特征生成器仅在训练时使用，因此 SimpleNet 在推理时遵循单一流式。这些模块将在下文按顺序描述。

其实特征提取它本质上是在做一个 “多尺度特征融合” 的工作，这部分和 PatchCore 几乎一模一样。SimpleNet 的创新点不在这个提取器（这是标准动作），而在于它后面加了一个 Adapter（适配器）（直方图 里蓝色的部分），通过引入高斯噪声来训练这个适配器，让特征分布更紧凑，从而不需要像 PatchCore 那样存一个巨大的 Memory Bank。
由于预训练网络偏向其训练的数据集，因此仅选择部分层级作为目标数据集是合理的。

特征适配器
由于工业图像通常与骨干预训练中使用的数据集分布不同，我们采用特征适配器 G 将训练特征转移到目标域。特征适配器可以由简单的神经块组成，如全连通层或多层感知器（MLP）。我们实验发现，单层全连通层性能良好。

异常特征生成
虽然[17， 18， 30]依赖额外数据合成缺陷图像，但我们在特征空间的正常样本上添加了简单噪声，声称其性能优于那些方法。异常特征通过在正常特征上添加高斯噪声生成。在区分异常特征与正常特征时，正常特征空间往往是紧致的，异常的会比较散开。

判别器
由于负样本与正常特征一起生成，训练时它们都会输入给判别器。判别器期望正常特征输出正输出，异常特征则输出负值。我们简单地使用两层的多层感知器（MLP）结构，像常见分类器一样。

函数异常特征生成器在推断时被舍弃。注意，剩余模块可以堆叠成端到端网络。我们依次将每个 x∈输入到上述特征提取器，并输入特征适配器，以获得公式 5 中的适配特征 qa。异常评分由判别器提供，对结果进行插值，使其具有输入样本的空间分辨率，并以高斯滤波，平滑边界以σ=4。由于异常区域的任何大小都存在最大响应点，异常图的最大得分即为每幅图像的异常检测得分

我们遵循单类分类协议，也称为冷启动异常检测，即对每个类别在其正常训练样本上训练一个单类分类器。

当主干为类 ResNet 架构时，主干的第二和第三中间层被用于特征提取器。默认情况下，我们的实现使用 WideResnet50 作为骨干网，特征提取器的特征维度设置为 1536。特征适配器本质上是一个无偏置的全连接层。适配器中 FC（全连接） 层的输入和输出特征尺寸相同。后续的判别器由线性层、batch归一化层、一个leaky relu层（斜率 0.2）和线性层组成。使用 Adam 优化器辅助学习特征适配器和判别器。

参考文献问题

致命错误 1：“others” 幽灵作者（最严重！）
你看你的 [56], [57], [83], [93], [124], [125] 这几条文献，作者列表里竟然出现了 others 这个词！

例如： [83] Tian J, others. RD++…

原因： 原始 BibTeX 里的作者写的是 author = {Tian, J. and others}。Zotero 直接把 “others” 当成了一个叫 Others 的外国人的姓氏！

怎么改： 这在盲审中是绝对的“低级错误”。你必须去 Zotero 里，把 others 删掉，补全前三个作者的名字。GB/T 7714 格式会自动帮你把超过三人的作者转换成正规的 et al.。

致命错误 2：缺失关键出版信息
你看 [29]：

[29] Shyu M ling, Chen S ching, Sarinnapakorn K, et al. A Novel Anomaly Detection Scheme Based on Principal Component Classifier[J]. 2003.

问题： 只有年份，没有期刊或会议名字！ 这是一篇发表在 IEEE ICDM 2003 上的经典异常检测论文。请去 Zotero 补上会议名称。

致命错误 3：大小写忽大忽小
这也是 BibTeX 导入的常见病。英文文章的标题（Title）在参考文献中，应该统一使用句首字母大写（Sentence case），而不能是所有首字母大写（Title case）。

ps. 页眉和页尾 选项卡那里的 “链接到前一条” 也要自己判断是否打开，如果打开，很多都会跟着前面走。一般情况 奇数页关 偶数页开。

如果页眉右侧的灰色标签 “与上一节相同”（Same as Previous）依然存在，说明你并没有成功取消链接。
在 Word 的逻辑中，只要这个标签还在，你在这一章修改页眉，前一章的页眉就会跟着一起变。

页码
