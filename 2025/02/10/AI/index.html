学习 YOLO

一个格子两个预测框，每个预测框用5 个参数，四个是 xywh，x 是预测框 bounding box 的中心点距离格子 cell 左边的距离，y 是中心点距离 cell 上边的距离（注意，一张图片分割出来的 49 个 cell 的格子都是一样大的），w 是预测框的宽度（和 x 同向），h 是高度，所以归一化的时候，是 x=x/wc，y=y/hc，wc 和 hc 分别指的是格子的宽度和高度，h 和 w 是整个图像的高度和宽度。
然后每个格子还需要一组类别值，注意只有一组，虽然是两个预测框，但他们共享一组类别概率值值，比如 VOC 数据集20 个类别，那就有 20 个类别概率值，分别代表这个格子属于这个类别的概率值，进一步说，使用这一组 20 个概率值来形容这个格子属于哪一个类。
不管一个单元格预测了多少个 bounding box，一个单元格也只能预测一组类别概率值。
对一张图片我们切割成 77=49 个格子，所以一张图片的预测结果一共是 77*30 个 tensor(张量、或者说参数)。

每个预测框都有两个参数，第一个是 Pr(Object) 表示这个框中是否有物体，也就是 0 和 1，第二个是这个框的 IOU，即交并比，预测框和 GT 框的交集大小除以并集大小，也就是一个预测准度。
测试的时候，我们将这二者还有刚刚那 20 个概率值的任一一个分别相乘来得到 这个预测框是这个类别的概率和框包含物体的准度。即 **特定类出现在 cell 内的概率和位置匹配程度(边框拟合程度)**，感觉不要太区分 cell 和 boundingbox 的区别。

最终，每个格子只能预测一个类别，所以限制了 yolo 预测重叠或临近物体的数量（比如鸟群中的鸟），yolo 只能预测 49 个对象。

每个 cell 虽然有两个框，但是是两个中取一个，框的粗细表示了可信度，选择可行度高的那一个。

表示这个 cell 预测是哪一个类

一个 cell两个框，框的中心点落在哪一个 cell 就由哪一个 cell 来预测这个框? 那万一有很多个框的中心点都落在一个 cell 里面呢？

每个框的置信度指的是 iou，暂时和类别概率是无关的。

1*1 的卷积核的话，是用来对特征进行降维的，比如一个 wh 的图像，他的特征可能有 D 个通道（不知道这样说是否准确），然后这个卷积核应该是 11D 的，卷积运算就是将卷积核 和 图像的每一个 D列对应相乘并相加，所以卷积结果是 wh的，如果想要结果的通道数不止为1，那就多用几个这样的卷积核，注意，卷积核通道数一定得和原图像一样。（这个通道应该是特征的通道，而不是什么rgb）

目前里接下来，一个cell值预测一个对象（即便对于重叠或很近的两个物体，也只能预测一个，预测的就是我们标注ground true box的时候标注的那个物体），所以不知道为什么要两个bounding box（打对抗？），然后如果对这个cell来说存在对象，那会由20个类别值来表示这个cell的对象应该是哪个类别，20个值加起来为1。这里说的这个cell的对象应该指的是，如果要预测的对象的中心落在这个cell里面，那就由这个cell负责这个对象的预测。

fast yolo 就是用到了更少的卷积层和滤波器，更快了。

训练的时候使用了 卷积层、平均池化层 average-pooling、全连接层 fully connected layer。
top 5 正确率，模型在数据集推理（什么是推理，看起来像是 “使用这个模型”，比如我们现在是使用这个模型进行测试）的时候，按照置信度排序生成5个标签，如果前五个标签有一个是正确的，则认为正确预测。

一般 cnn 分成卷积层（用来特征提取）和全连接层（用来分类）。当然卷积层有很多不同的特征提取层，比如有的用来提取点和线，有的提取边和角，有的用来提取语义信息（比如人啊动物啊这样的）。

卷积

卷积就是拿一个卷积核（模板）去找，让他匹配计算，如果匹配到一个和模板很像的区域，就会得到一个大值。
卷积结果叫特征图 feature map（或者叫特征图像），是对图像是否包含这个特征的一种表示，包含的话包含在哪，那就是比较大的那个值。特征图像是小于等于原始图像的。所以想要有效果，卷积核一定要好好设置。

全连接层

用来提取特征，举例子这里通过卷积提取了 H、O两个特征，全连接指的就是前面的H、O和和后面的 H2O、O2、H2 是全连接上的（每一个神经元和下边的神经元都是连接上的），并且全连接中还有权重，有时候我们会进行 dropout（随机失活，让一些神经元不可用，来防止过拟合）。

池化
池化和卷积实际比较像，本质都是一个模板（比如在卷积里就是卷积核）进行滑动，池化的作用是将相似的特征进行结合，从而达到降维的效果。

NMS (Non-Maximum Suppession) 非极大值抑制

一般是一个 cell 预测一个对象，但可能出现多个 cell 预测同一个对象，可能是对象的中心在多个cell 的边界上。我们使用 NMS 来解决这个问题，提升了 23的精度（mAP）。

有三个框，但通过抑制只剩一个，筛选出目标得分最多的框，达到去重。

选出分（置信度）最高的框，然后删除它以及和它很相近的框（比如交并比iou很大的，大于某个范围的那些框），依次进行这个步骤。

缺点：

对 boundingbox 加了很强的空间约束
每个cell只预测两个bbox，且只能有一个类别。
空间的约束限制了模型可以预测的邻近物体的数量，对鸟群中的鸟就比较困难。
模型从数据中学习，无法泛化到新的没见过的对象。

由于模型中包含多个下采样层（大图像变成小图像，比如池化），所以模型使用的特征也比较粗糙，丢失了精度。

　(1)True positives(TP): 被正确地划分为正例的个数，即实际为正例且被分类器划分为正例的实例数（样本数）；

　　(2)False positives(FP): 被错误地划分为正例的个数，即实际为负例但被分类器划分为正例的实例数；

　　(3)False negatives(FN):被错误地划分为负例的个数，即实际为正例但被分类器划分为负例的实例数；

　　(4)True negatives(TN): 被正确地划分为负例的个数，即实际为负例且被分类器划分为负例的实例数。
1.精确率（precision）

precision = TP / ( TP + FP )

精确率是精确性的指标，表示被分类器正确分为正例的个数(TP)占被分类器分为正例的样本(TP+FP)的比重。

2.召回率（recall）

recall = TP / ( TP + FN ） = TP / P

召回率是覆盖面的度量，也就是被分类器正确分为正例的个数(TP)占原始数据中全部正例(TP+FN)的比重。

只有当P和R都很高的时候，F1(用准确度和召回率算的)才会高，所以称为调和平均数，F1的取值范围是0到1。

5.accuracy

准确率，这个跟精确率只有一字之差，但实际上有很大的不同，它是指正确分类（不管是正确分为P还是N）的比率

accuray = （TP + TN) / (TP + FP + TN + FN ) = (TP + TN) / (P+N)

Precision衡量你的预测有多准确。也就是说，你的预测正确的百分比。
Recall衡量您发现所有正例的能力。 例如，我们可以在前K个预测中找到80％的正例。（TopK）

AP 平均精度 就是 PR 曲线的面积。mAP 就是 所有类别的 AP 的平均值。
为什么要用AP或者mAP
我们希望一个模型的Precision和Recall都很高，所以需要综合考虑这两个因素，我们可以联想到用调和平均数F1-beta值来衡量，另一种方法正是PR曲线下的面积AUC，这也就是AP。AUC面积越接近1性能越好。曲线下的面积理解为不同召回值的情况下所有精度的平均值。

自注意力机制
ViT不直接看像素，而是对图像进行切分，比如224*224的切成14*14个方块，每个方块叫token，每一层中，每个方块会向所有方块询问，比如最左边一个快是垫片，他询问谁和我有逻辑关系。，此时图像最右边的token回答我是螺栓和你是一套的，因此即便距离很远，模型给他们之间连上高权重的线，从而信息交换。而CNN要实现这个最左最有的通信可能需要几十层的卷积，中间信息模糊了。
在铁路场景，可以借助长距离依赖来实现对称性检查，比如左右各一个弹簧，左边坏了，CNN很难通过对比来发现异常，而ViT 同时看两个，发现二者不一样。 此外如果我们要求物件在某个背景下才算是正常，自注意力也可以把背景信息编码进零件的特征。
ViT具有全局视角

卷积核通常很小，感受野很小，不下采样的话，卷积后的模型始终只能看到局部，下采样相当于退后一步看全身，减少图像尺寸，同样的卷积核能覆盖到原图更大比例的面积。同时也能减少计算压力，去除冗余信息。
下采样通常通过池化或者步长卷积来实现，池化比如最大池化，步长卷积指的是卷积核跳着走，比如步长2，扫完一个区域之后跳过一个像素再扫，这样特征图长宽就减半了。

我们使用跳跃连接把分辨率救回来，它相当于从前端（浅层特征，知道位置，但不明白语义）即哪些还没模糊的高清特征图直接投送到后端（深层特征，具备语义，你知道这个点有螺栓那里有裂纹但你不知道具体位置）。
具体是，将深层7*7的特征图上采样（反卷积或者插值）便会224*224特征图，虽然够大但是模糊，然后这张图和那张浅层图叠在一起，浅层图告诉模型这里有个东西（位置信息），深层图告诉模型这个是什么（语义信息）。

反卷积：指的是有参数的填充放大，将小尺寸特征图填充0放大，然后学习一个卷积核去扫这张撑开的图，学习去用合理的数值填充这些0，使得结果清晰又合理，由于卷积核带参数，效果比简单的插值放大更合理。

FPN 特征金字塔就是跳跃连接的升级版，能同时检测巨大的形变和微小的划痕，是目标检测和异常检测的关键。
区别在哪，跳跃连接 U-Net 通常只在最后的一层高分辨率特征上进行预测，应该是说 低分辨率 特征上采样之后和高分辨率特征融合，只用这一层进行预测。而FPN 是多层预测，在金字塔的每一个层级都分别进行检测，比如 p2 层专门检测10像素的小裂纹，而p5层专门检测半张图的大型变形。还提到“似乎每一层的信息是实时共享的”？如何理解

FPN 的特点应该是，它在每一层特征都做了跳跃连接。
FPN 的构造分成三个路径，第一个 自下而上，是特征提取过程，就是之前说的下采样，特征图分辨率越来越低，语义越来越强，能认得出东西。 第二个是 自上而下，把深层特征进行上采样放大，试图让他回到高分辨率的状态。 第三个是 横向连接，他将第一步路径中 每个尺寸的特征图通过 1*1 卷积处理后 直接加到 第二步路径中的特征图，从而在每一个尺度上，都生成了一个 具有高清分辨率和深度语义的特征图。
那么 FPN 它的特点就是，它的顶层特征（感受野大）能去识别大物体，底层特征（分辨率高）能去识别小的痕迹。
通过多层特征融合，增强了模型对噪声的鲁棒性，即使某一层特征不明显，其他层的融合信息也能一定程度上起到补位的作用。
还提到一个 语义对齐，比如我们要判断某个小物体 螺栓 不仅正常，也处于某个正确的位置，没有松动，我们需要模型唔够获取周围的结构信息，这就属于 深层语义，FPN 的横向连接就是让每一个小尺度特征都带上 全局的“脑子”，让它能获取不仅限于这个小尺度的语义信息？

上面提到的 1*1 卷积核，他又很大的作用，第一个 对齐通道数，降为或升维，自下而上，浅层（分辨率高）通道数小，深层的通道数可能到了512，如果希望将两张特征图叠加，那要求通道数必须一致，1*1 卷积核就能将512的深层特征压缩为64通道，将同一个位置不同通道的信息进行组合。

再讲一遍：
第一阶段 自下而上提炼特征：p2浅层尺寸大（注意理解这个尺寸），分辨率高，语义差，升级到p5深层，尺寸小（下采样很多次），能看到整体布局，认出这个螺栓应该在的位置（在转向架的左边），也就是语义强，但是它看不清楚螺栓，只是知道它在那里。
第二阶段 自上而下传递智慧，将p5深层特征反卷积放大到和p4一样大，此时，这张大图仍然具有较为强大的语义信息，具备“全局意识”。
第三阶段：一对一融合，将放大后的p5（和4一样大，但具有5的大局观）和原来的p4（有原来的细节）进行像素级相加，融合得到的特征图，每一个像素点既包含原来p4的高分辨率位置细节也包含了p5传下来的语义信息。

举个例子：假如我在p2（分辨率很高的一层），我现在看到一个黑点，我认不出是裂纹还是油渍，但是有了FPN，深层（p5、p4、p3）的语义信息一层一层向下传给了p2，现在p2的哪个点，他携带的信息就成了“我是位于 5：转向架、4：左侧第三个 3：螺栓上的裂纹”，从而获取到不属于自己尺度的语义信息，它们从金字塔顶端顺着 自上而下 的路径留下来，和每一层的局部细节汇和。
